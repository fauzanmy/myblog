Melalui update Core Web Vital, google merekomendasi format gambar WEBP untuk konten blog. Secara default aplikasi Photoshop yang bisa saya gunakan untuk mengedit gamabar, tidak mendukung Save As ke WebP.

Tapi, berkat plugin khusus, Photoshop saya sudah bisa menyimpan gambar ke format .webp.

Plugin apa saja yang cocok dengan Adobe Photoshop anda, baca saja ulasan panjang artikel berikut https://breedie.com/ubah-jpg-ke-webp-dengan-photoshop/.
